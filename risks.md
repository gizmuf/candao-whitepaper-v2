# Risks & Disclaimers

> **This Whitepaper presents work in progress. Final products may differ.**

## General Risks

| Risk Category | Description |
|---------------|-------------|
| **Regulatory** | Crypto/AI regulation evolving; may require operational changes |
| **Technology** | AI performance varies; smart contracts may have vulnerabilities |
| **Market** | Token prices fluctuate; liquidity may be limited |
| **Execution** | Timelines are estimates; delays possible |

## AI-Specific Risks

| Risk | Mitigation |
|------|------------|
| Suboptimal Ghost decisions | Human-in-the-loop for high-stakes actions |
| Unpredictable AI behavior | Action logging, permission boundaries |
| Failed negotiations | Fallback to human interaction |
| Data exposure | End-to-end encryption, user-held keys |
| **Permission misuse** | Ghosts cannot exceed user-set limits; all actions auditable |
| **Ghost impersonation** | Blockchain-verified Ghost identity; cryptographic signatures |
| **Unauthorized transactions** | Multi-factor for high-value; instant revocation available |

**User Responsibility:** Users are ultimately responsible for setting appropriate Ghost permissions and monitoring Ghost activity. Candao provides tools and defaults for safety, but cannot prevent all misuse.

## Economic Model Risks

| Risk | Mitigation |
|------|------------|
| Zero-fee sustainability | Partner revenue + ecosystem fees fund operations |
| Token value dependency | Diversified treasury; not sole funding source |
| Buyback manipulation | DAO governance, third-party audits, timelocks |
